# 재난 및 안전시설 모니터링 자동화 도구

## 사용 AI 도구
- Gemini 3.5 Flash & Claude Code
- 코드 작성, 데이터 파싱 자동화 및 다국어 인코딩 문제 디버깅

## 구현 메모
- **의존성 있는 분석 수행**: pandas 라이브러리를 사용하여 대규모 CSV 파일의 수동 로드 없이 자동으로 통계 지표(평균, 합계, 빈도 등)를 집계
- **데이터 변환 및 무결성 확보**: CSV 데이터를 Javascript에서 파싱하기 쉽도록 JSON 파일(`data.json`)로 가공. 이 때 Pandas의 결측치(`NaN`)가 JSON 내에서 잘못 파싱되지 않도록 파이썬 `None`으로 정교하게 매핑 처리
- **동적 상태 조건 시각화**: 웹 대시보드에서 피해액 3억 원 이상(30,000만 원)에 해당하는 중대 재난 항목에 동적으로 클래스를 적용하여 시각적 테두리 경고등 효과 제공
- **인코딩 표준 준수**: Windows 개발 환경의 콘솔(CP949) 및 웹 브라우저 간의 텍스트 깨짐을 방지하기 위해 입출력을 UTF-8 파일 인코딩으로 고정하고, 파이썬 표준 스트림 출력을 UTF-8로 강제 재조정
- **RSS 연동 및 장애 대비**: RSS 피드 로드 오류(네트워크 순단 또는 CORS 제약) 상황에서도 사용자 경험이 훼손되지 않도록 양질의 백업 정책 데이터를 스크립트에 탑재

## 실행 결과
- **총 재난 모니터링 건수**: 500건
- **심각 등급 재난 (4-5등급)**: 285건 (전체의 57%)
- **누적 피해액**: 122,053,610,000원 (약 1,220억 원)
- **평균 피해액**: 249,089,000원 (약 2억 4,900만 원)
- **등록 안전시설**: 100개
- **안전시설 점검등급 분포**: A등급 35개, B등급 39개, C등급 17개, D등급 9개
