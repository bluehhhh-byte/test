# 국가 재난 및 안전 모니터링 시스템 (Disaster & Safety Monitoring System)

재난 발생 현황과 전국 안전시설 현황을 한눈에 모니터링할 수 있는 대시보드 웹페이지와 데이터 요약 분석을 위한 파이썬 자동화 툴킷입니다.

## 🛠️ 주요 기능 및 아키텍처

1. **인터랙티브 웹 대시보드 (`index.html`)**
   - **실시간 대시보드 레이아웃**: 메트릭 카드(총 건수, 피해액, 심각 단계 건수, 안전시설 수) 시각화.
   - **재난 발생 카드**: 재난유형별 다채로운 배지 색상 구분 및 피해금액 단위 가독성 개선 (`억`, `만` 원 단위 표시).
   - **동적 가시성 제어 (Dynamic Status Glow)**: 피해액 3억 원 이상인 대규모 피해 사건은 경고 배지와 함께 붉은색 그림자 효과가 적용됩니다.
   - **실시간 필터링**: 재난 유형 및 발생 지역을 조합하여 실시간으로 필터링하고 검색 결과 건수를 갱신합니다.
   - **안전시설 동적 집계**: 시설유형별/지역별 안전시설 현황을 자바스크립트로 계산하여 막대그래프 형태로 비율을 시각화합니다.
   - **행정안전부 RSS**: `korea.kr` 정책 및 보도자료 피드를 비동기 로딩하며, 오류 시 백업 뉴스를 연동합니다.

2. **통계 요약 프로그램 (`summary_tool.py`)**
   - Pandas 라이브러리를 활용하여 두 개의 원본 CSV 파일에서 재난 발생 현황과 안전시설 배포 등급을 집계 및 연산하여 콘솔에 요약 출력합니다.
   - Windows 터미널 한글 깨짐 및 이모지 Unicode 출력 오류를 대비하여 `sys.stdout` UTF-8 재배치 코드를 탑재했습니다.

3. **데이터 포맷 변환기 (`convert_data.py`)**
   - 원본 CSV 파일을 읽어 웹 대시보드에서 활용하기 좋은 규격화된 `data.json` 데이터 세트로 저장합니다.

## 📂 파일 구성
- `index.html`: 대시보드 웹페이지
- `data.json`: 변환 완료된 재난/시설 원본 JSON 데이터
- `summary_tool.py`: 파이썬 통계 요약 프로그램
- `convert_data.py`: CSV to JSON 변환 프로그램
- `배포_URL.txt`: Vercel 배포 완료 경로가 기록된 파일
